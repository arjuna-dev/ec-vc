## 1) Description
In the shared create/edit record dialog, existing selected values are not rendering inside the input boxes when editing a record.

## 2) Environment
- Git branch: `ECS_Ingestion_Sandbox`
- Operating system: Windows
- App/runtime: Electron Quasar desktop app
- App version: Unknown

## 3) Expected Behavior
When opening the shared create/edit dialog for an existing record, the input fields should show the current saved selections and values for that record.

## 4) Actual Behavior
The dialog opens, but selected values are missing from the inputs, especially for selection-based fields.
