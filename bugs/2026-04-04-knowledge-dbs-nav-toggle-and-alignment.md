## 1) Description
The new `Knowledge DBs` grouping in the left navigation is not behaving correctly. Its expand/collapse interaction is broken/confusing, and the child labels are misaligned compared with the group title.

## 2) Environment
- Git branch: `ECS_Ingestion_Sandbox`
- Operating system: Windows
- App/runtime: Electron Quasar desktop app
- App version: Unknown

## 3) Expected Behavior
The `Knowledge DBs` row should expand and collapse like the other navigation groups. Its child rows should read `- Industries` and `- Securities`, and their text should start at the same horizontal point as the `Knowledge DBs` title. The group icon should match the surrounding nav rows in alignment and sizing.

## 4) Actual Behavior
`Knowledge DBs` does not open/close correctly, the child labels are not named as intended, the child rows are indented too far, and the group icon sizing/alignment does not match the surrounding navigation rows.
