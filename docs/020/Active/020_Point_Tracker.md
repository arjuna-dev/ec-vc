# Point Tracker

## Purpose

The `Point Tracker` maintains score, progress, and verification checkpoints for the game layer.

It ensures scoring reflects verified work and does not drift from actual completion status.

## Core Responsibilities

- track point awards, deductions, and verification status
- align scoring events with provenance/history entries
- surface scoring gaps or unclear rules before new points are awarded
