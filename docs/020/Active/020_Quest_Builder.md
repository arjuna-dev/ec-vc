# Quest Builder

## Purpose

The `Quest Builder` defines and curates quests, tasks, and progression logic for game loops.

It ensures quests map to real file and token outcomes instead of abstract tasks.

## Core Responsibilities

- define quest steps that map to explicit file-owned tokens
- keep quest requirements aligned with the current intake and record contracts
- flag any quest that would require legacy scaffolding or fallback behavior
